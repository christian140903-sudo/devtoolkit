# DevToolkit — Free Online Developer Tools

**8 free browser-based tools** for developers, designers, and creators. No sign-up, no tracking, works offline.

[Try it live](https://christian140903-sudo.github.io/devtoolkit)

---

## Tools

| Tool | Description |
|------|-------------|
| **Invoice Generator** | Create professional PDF invoices instantly |
| **QR Generator** | Generate QR codes for URLs, text, WiFi, vCards |
| **JSON Formatter** | Format, validate, and minify JSON data |
| **Password Generator** | Create secure passwords with custom rules |
| **Color Palette** | Generate and explore color palettes |
| **Meta Tag Generator** | Create SEO meta tags for your pages |
| **Markdown Editor** | Write and preview Markdown in real-time |
| **Word Counter** | Count words, characters, sentences, and reading time |

## Features

- 100% client-side — your data never leaves your browser
- No sign-up or account required
- Works offline after first load
- Mobile-responsive design
- Fast and lightweight

## Tech Stack

- Pure HTML, CSS, and JavaScript
- No frameworks, no dependencies
- Hosted on GitHub Pages

## Usage

Visit the [live site](https://christian140903-sudo.github.io/devtoolkit) or clone this repo and open `index.html` in your browser.

```bash
git clone https://github.com/christian140903-sudo/devtoolkit.git
cd devtoolkit
open index.html
```
